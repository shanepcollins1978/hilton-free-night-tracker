# Hilton Free Night Tracker

A mobile-friendly browser app for tracking Hilton Amex free night spending progress.

## Included trackers

- Shane’s Hilton Aspire, $60,000 goal, accounts -41008 and -41016
- Shane’s Hilton Surpass, $15,000 goal, accounts -22005 and -72011
- Diana’s Hilton Surpass, $15,000 goal, accounts -71005 and -21031

## Features

- CSV import
- Automatic account routing
- Duplicate transaction prevention
- Collapsible transaction lists
- Local browser storage
- Mobile home screen support
